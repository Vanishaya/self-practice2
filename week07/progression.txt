1.เลือก <div id="demo"> จากหน้า HTML เพื่อจัดการใน JavaScript innerHTML อ่าน “HTML ทั้งหมดภายใน” รวมแท็กและข้อความ 
innerText อ่านเฉพาะ “ข้อความที่มองเห็นบนหน้าเว็บ” ไม่รวมที่ถูกซ่อน textContent	อ่าน “ข้อความทั้งหมด” รวมถึงที่ซ่อน เบราว์เซอร์จะตีความแท็ก <i> แสดง ตัวเอียงจริงๆ บนหน้าเว็บ
แสดง <i>Sample Italic Text</i> ไม่เอียง แสดง <i>Sample Italic Text</i> เหมือน innerText
แต่ต่างกันตรงที่ textContent จะอ่านข้อความที่ถูกซ่อนไว้ได้

2. เขียนโค้ดแสดง ชนิดของ Node ที่อยู่ภายใน <div>
เช่น Element node, Text node และแสดงค่า nodeName, nodeType, nodeValue ของแต่ละ node

3.เขียนโค้ดที่แสดงเฉพาะ ข้อความที่อยู่ใน Text Node ของ element (ข้าม element node เช่น <span> หรือ <b>)

4.สร้าง <div> ที่มี attribute เช่น id และ style
จากนั้นแสดงชนิดของ Node ที่เป็น Attribute (Attr Node) และ Element (Element Node)
